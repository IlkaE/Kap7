Suchformular für Wikipedia

Mit diesem Formular können Sie im Internet-Lexikon Wikipedia suchen.

Geben Sie dazu in das Feld "Suchwort" einen Begriff ein, den Sie im Lexikon nachschlagan möchten.

Beim Klicken auf Artikel wird ein Lexikon-Artikel gesucht, der diesen Begriff zum Thema hat.
Beim Klicken auf Volltext werden alle Lexikon-Artikel gesucht, in denen dieses Wort vorkommt.
